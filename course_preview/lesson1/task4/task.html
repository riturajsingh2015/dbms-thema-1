Aufgabe Beschreibung- Task4

Hinweis : Dies ist eine Ausgabeaufgabe, bei der die Ausgabe des Programms mit der Ausgabe in der Datei output.txt verglichen wird, wenn sie gleich ist, erhalten Sie als Ausgabe Glückwünsche. Beachten Sie auch, dass beim Klicken auf den Check-Button auch hier die main-Methode ausgeführt wird, was bei Testaufgaben nicht der Fall ist.

task4 ~ erstellt einfach eine Tabelle namens users mit den Attributen (id, name , password & age)

Die Aufgabe besteht darin, die Lücken "DROP", "CREATE" mit den richtigen Antworten zu füllen, wenn Sie fertig sind, erhalten Sie als Antwort Glückwünsche.

DROP : Die DROP-Abfrage sollte die Tabelle namens users löschen, wenn sie bereits vorhanden ist.
CREATE : Die Create-Abfrage sollte eine Tabelle namens Benutzer mit den Attributen (id , name , password, age) erstellen, wenn sie nicht bereits vorhanden ist.
und mit folgenden Typen :
id int, name varchar(200),password varchar(200),age int
