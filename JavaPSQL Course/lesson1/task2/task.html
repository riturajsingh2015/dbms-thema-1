Aufgabe Beschreibung- Task2

Hinweis : Dies ist eine Testaufgabe, daher ist die Lösung dieser Aufgabe auch in Peek-Solutions verfügbar

task2 ~ Da Postgres komplexe Datentypen wie "bytea" erlaubt, die zum Speichern von Bildinhalten verwendet werden können

Daher beschäftigt sich diese Aufgabe damit, wie wir ein Bild in unserer Datenbank speichern können.

Die Aufgabe besteht darin, die Lücken "DROP", "CREATE" und "INSERT" mit den richtigen Antworten zu füllen, wenn Sie fertig sind, erhalten Sie als Antwort Glückwünsche.

DROP : drop query sollte eine Tabelle mit dem Namen "images" löschen, falls sie bereits vorhanden ist.
CREATE : create query sollte eine Tabelle mit dem Namen "images" und mit den Attributen "id serial, data bytea" erstellen, falls sie nicht existiert.
INSERT : Die Einfügeabfrage sollte das Einfügen des Inhalts des Bildes in das Attribut "data" ermöglichen
