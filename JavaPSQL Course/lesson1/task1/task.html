task1 : beschäftigt sich damit, wie wir mit JDBC als API eine Verbindung zur Postgres-Datenbank herstellen können.

AUFGABE ~

Die Aufgabe besteht darin, die Lücken mit "url", "user" und "password" auszufüllen. Und wenn Sie die Lücken richtig ausgefüllt haben, können Sie Ihre Antworten auf ihre Richtigkeit überprüfen.

In dieser Aufgabe können Sie lernen, was eine JDBC-URL ist und wie man sie mit der java.sql-Bibliothek verwenden kann, um die Verbindung mit der Datenbank herzustellen. Und wenn Sie die Lücken richtig ausgefüllt haben, können Sie Ihre Antworten auf ihre Richtigkeit überprüfen.

Um eine Verbindung herzustellen, müssen Sie eine Connection-Instanz von JDBC erhalten. Dazu verwenden Sie die Methode DriverManager.getConnection():

Connection db = DriverManager.getConnection(url, username, password);

Der Hostname des Servers : localhost
Name unserer Datenbank : test
Die Port Nummer-Standardmäßig : 5432
